Galileo is unfinished, in progress. It is a program that implements clustering algorithms we're working on...

  • To install, sign on to R, and type: install.packages("Galileo",repos="http://gking.harvard.edu")
  • For documentation, from R, type library(Galileo), and then ?galileo, or download PDF (or try the translated html version)
